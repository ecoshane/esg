關於我們

Ecoshane 永續家是 Ecoshane Bank 旗下的專業顧問部門。我們的使命是協助企業將環境（E）、社會（S）的投入與風險管理，轉化為可審核、可量化的環境與社會績效指標 (E&S SPTs)。我們致力於在永續策略（包含風險管理）與金融執行之間建立高效、透明的溝通橋樑。

社會價值貨幣化 (SROI)

我們的 SROI 評估模型完全遵循英國 SVI (Social Value International) 的七大原則框架進行精確評估。確保將人力資本投入、公正轉型成本等社會效益，轉化為國際認可、可審核的貨幣價值，直接作為永續連結貸款（SLL）的社會 SPT 依據。

E&S 數據治理與確信

利用 Open API 和雲端技術，建立環境績效（如碳排）與社會風險（如申訴案件）數據的自動化驗證機制。為企業取得國際會計師事務所有限確信報告，確保最高的透明度與信賴度。

國際公信力基礎：英國 SVI 框架

我們的社會投資報酬率 (SROI) 計算與報告方法，嚴格依循 Social Value International (SVI) 所制定的七大原則框架。確保所有評估結果在國際上具備最高度的透明度、嚴謹性與公信力。

核心服務：創造 ESG 的貨幣價值

永續連結貸款 (SLL) 輔導

為企業量身設計環境 (E) 與社會 (S) 雙軌 SPT 目標，確保達成指標時，能自動觸發貸款利息減免，優化企業融資成本。

SROI 社會價值評估 (SVI 框架)

我們採用英國 SVI 的國際公認方法論，將員工培訓、社區投入等社會專案，量化為具體的貨幣效益。此評估結果具備高度公信力，可直接納入 SLL 績效評估。

人權盡職調查 (HRDD) 導入

協助企業建構人權風險管理框架 (UNGP)，包括鑑別潛在風險、建立營運層級申訴機制，並輔導撰寫人權政策，滿足國際法規要求。

供應鏈 E&S 風險管理

輔導企業建立供應鏈稽核與風險分級制度，特別針對強迫勞動與童工等重大違規事項，建立即時預警與改善機制，確保供應鏈韌性。

E&S 數據治理與 API 串接

輔導企業導入標準化數據介面 (Open API)，實現碳排、能耗數據，並整合勞工申訴平台與供應鏈稽核的風險數據，簡化審核流程。

碳盤查與減碳策略

協助企業進行溫室氣體盤查 (ISO 14064-1)，提供查證輔導，並制定符合科學基礎減量目標 (SBTi) 的減碳策略。

永續報告書撰寫 (GRI/SASB)

專業團隊協助企業撰寫符合主管機關規定的永續報告書，並進行內部查核與預審，確保報告順利通過第三方查證。

綠色轉型策略 (TCFD)

提供生產流程綠色轉型方案，協助導入再生能源、循環經濟模型，並進行氣候變遷相關財務揭露 (TCFD) 輔導。
